# Profile Card UI Design

A modern and responsive **Profile Card UI** built using **HTML5 and CSS3**.  
This project showcases a clean personal profile card with social media links, hover animations, and a professional layout suitable for portfolios and personal websites.

---
## 📸 Preview
![Profile Card Screenshot](screenshots/profile-card.png)

---
## ✨ Features
- Responsive profile card layout
- Circular profile image with border
- Smooth hover animations and transitions
- Social media icons using Boxicons
- Call-to-action buttons (Subscribe & Message)
- Clean and modern UI design

---
## 🛠️ Technologies Used
- HTML5  
- CSS3  
- Google Fonts (Poppins)  
- Boxicons Icons
